

# Movie Hub
Movie Hub is a desktop application used to play and organise your locally stored movies.

### Features

- Movie decoder based on VLC.
- Internet radio with 2000 stations available.
- Themes.

### Prerequisites
Windows requirement .Net Framework >= 4.0
